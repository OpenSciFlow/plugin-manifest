# Security model

OpenSciFlow plugins can describe local commands. This creates real security risk.

## v0.1 safety assumptions

- The user explicitly approves plugins before execution.
- Commands come from reviewed manifests, not arbitrary LLM output.
- Paths are resolved inside approved work directories.
- Model weights and containers are downloaded only after approval.
- Every run records rendered commands and environment metadata.

## Do not do this

- Do not execute arbitrary shell generated by an LLM.
- Do not auto-install unknown packages silently.
- Do not download model weights without license/source information.
- Do not upload user data to cloud by default.

## Local Agent expectations

The Local Agent should:

- execute only allowlisted plugin commands;
- stream logs;
- compute input/output hashes;
- reject paths outside approved directories;
- return artifact metadata;
- keep private data local by default.
